# @jinpill/react-libs

Jinpill's personal react libraries.

## Installation

```bash
# npm
npm i -D @jinpill/react-libs

# yarn
yarn add -D @jinpill/react-libs

# pnpm
pnpm add -D @jinpill/react-libs
```

Import and use the components or hooks in your project.
